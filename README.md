<h1>want to run it locally?</h1>

create a archive .env

and put

```
TOKEN=yourtoken
```

and finally run:

```
deno run -A -q --unstable mod.ts
```

or if you want to run it with docker just run:

```
sudo docker build -t deno-bot . && sudo docker run -d deno-bot

```
